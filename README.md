# 🚀 PitchPal – Your Hackathon AI Agent for Pitch-Perfect Presentations  

PitchPal is a **SaaS-powered AI Agent + Full Stack** built for hackathons and startups. From **PPT generation** to **summary creation**, PitchPal helps you craft **investor-ready pitches in minutes** — so you can focus on **building, not stressing**.

✨ Whether you're in a **24-hour hackathon** or pitching your **startup idea**, PitchPal ensures your deck is sleek, professional, and impactful.  

---

## 🌟 Features  

- ⚡ **AI-Powered Slides** — Generate sleek & customized pitch decks instantly.  
- 🧠 **Smart Summarizer** — Condense your idea into clear, crisp narratives.  
- 👥 **SaaS for Teams** — Cloud-based, collaborative & hackathon-friendly.  
- 💳 **Seamless Payments** — Credits are added instantly after Razorpay payment. Use them across all our AI tools without interruptions.  
- 🤖 **AI-Powered Tools** — Access powerful AI tools built for smooth workflows — from content generation to smart automation.  
- 📊 **Smart Insights** — AI-driven analytics, summaries, and instant feedback to guide better decision-making.  
- 🔄 **Streamlined Workflow** — Save time with automation and integrated tools.  
- 💾 **Auto Save & History** — Your work is automatically saved and maintained, so you can revisit and continue anytime.  
- ✨ **Export & Share** — Download, export, and share results instantly with built-in export features for complete flexibility.

## 🛠️ Tech Stack  

- **Frontend**: Next.js, React, TailwindCSS, Framer Motion  
- **Backend**: Node.js, Express  
- **Database**: MongoDB  
- **Payments**: Razorpay  
- **AI**: OpenAI API + Custom AI Models  
- **Deployment**: Vercel / Render
